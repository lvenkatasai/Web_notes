Quick User Guide

Getting Started

Click "Add Note" to create a new note.
Fill in title, content, tags, and choose a color.
Notes are automatically saved to your browser.

Editing Notes

Click the edit icon to change a note.
Click the delete icon to remove a note.
All changes are saved immediately.

Organizing Notes

Use tags to categorize your notes (e.g. "work").
Color-code notes by visual preference.
Search notes by title, content, or tags.

Data Management

Export all notes as a JSON file for backup.
Import notes from a JSON file when needed.
Use "Clear All" to delete all notes permanently.

Quick Tips

  • Notes are saved as you type
  • Newest notes appear first
  • Use descriptive tags for better organization
  • Search results update in real-time
  • Export your notes regularly as backup
